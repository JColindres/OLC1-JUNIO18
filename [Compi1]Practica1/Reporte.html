0 Declaracion de decimal Declaracion de variables que sirve para almacenar valores de tipo decimal.

4 Sentencia Si Sentencia para ejecutar acciones si la condicion es verdadera.
2 Sentencia Sino Sentencia para ejecutar acciones si la condicion de la sentencia si es falsa.
6 Sentencia Para Sentencia para ejecutar acciones un numero determinado de veces, tiene una inicializacion, una finalizacion y un incremento.
1 Sentencia mientras Sentencia para ejecutar acciones mientras cumpla con una condicion.
1 Sentencia Hacer-Mientras Sentencia para ejecutar acciones al menos una vez y es reejuctada cada vez que la condicion se evalua a verdadera.
2 Sentencia Seleccionar Sentencia que se utiliza para agilizar la toma de decisiones multiples.
2 Sentencia Continuar Es un tipo de control de bucles. Esta sentencia se salta al siguiente ciclo de la iteracion.
3 Sentencia Interrumpir Es un tipo de control de bucles. Esta sentencia rompe la iteracion de dicho bucle.
